1- Create - метод create используется для создания новой таблицы в базе данных.
2- Insert - это SQL оператор кооторый позволяет дабавлять строки в таблицу.
3- Select - это	SQL оператор который возвращает данные из базы данных.
4- Update - это SQLератор который позволяет обновлять значения в строках базы данных.
5- Drop - это SQL оператор который позволяет удалять строки из базы данных,при этом н удаляя саму таблицу.
